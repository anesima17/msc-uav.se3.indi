# Literatura i referentni materijali

Glavna literatura projekta:

1. Lee, Leok i McClamroch — Geometric Tracking Control of a Quadrotor UAV on SE(3)
2. Smeur, Chu i de Croon — Adaptive Incremental Nonlinear Dynamic Inversion
3. Bouabdallah — Design and Control of Quadrotors
4. Faessler i Fontana — Theory and Math Behind RPG Quadrotor Control
5. Lapandić — Disturbance-Aware Motion Planning and Control
6. Foehn et al. — AlphaPilot: Autonomous Drone Racing
7. MSc tema — Integrated Detect-and-Avoid for Indoor UAVs

PDF datoteke mogu biti sačuvane lokalno na Windows računaru kako bi ih
Claude Code mogao analizirati.

Referentni C kod `controller_indi.c` nije početna implementacija projekta.
Koristi se kasnije za poređenje teorije INDI-ja s praktičnom implementacijom.
