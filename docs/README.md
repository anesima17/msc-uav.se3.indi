# Dokumentacija projekta

Ovaj direktorij sadrži teorijsku i tehničku dokumentaciju MSc projekta:

**Integrated Detect-and-Avoid for Indoor UAVs:
Real-Time Trajectory Optimization and Robust Nonlinear Control**

Glavna ciljna arhitektura upravljanja sastoji se od:

1. geometrijskog upravljanja za vanjske petlje pozicije, brzine i
   formiranje željene orijentacije;
2. INDI regulatora u brzoj unutrašnjoj rotacionoj petlji;
3. raspodjele upravljanja na pojedinačne motore;
4. kasnije integracije planiranja trajektorije i detect-and-avoid sistema.

Planirani dokumenti uključuju:

- `project_scope.md`
- `learning_roadmap.md`
- `literature_map.md`
- `notation.md`
- `frames.md`
- `quadrotor_model.md`
- `geometric_control.md`
- `indi_control.md`
- `controller_integration.md`
- `assumptions.md`
- `open_questions.md`
- `decisions.md`

Dokumenti će se izrađivati postepeno. Ne treba unaprijed kreirati prazne
fajlove bez sadržaja.
